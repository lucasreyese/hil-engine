# Python Simulator
This simulates a 1.8L fixed-transmission engine with variable crankshaft speed under a constant load. Physics-based models derive sensor outputs from engine state, forming a closed loop with the ECM.

These sensors include:
- RPM (Revolutions Per Minute)
- TPS (Throttle Position Sensor)
- MAP (Manifold Absolute Pressure)
- ECT (Engine Coolant Temperature)
- IAT (Intake Air Temperature)
- Knock (Vibration sensor)
- O2/Lambda (Oxygen sensor, post-combustion)
